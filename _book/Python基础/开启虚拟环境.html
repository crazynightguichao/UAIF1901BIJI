第3节：开启虚拟环境

开启虚拟环境（局部环境）（会污染全局）

python -m venv tutorial-env （创建虚拟环境，多了一个文件夹）

tutorial-env\Scripts\activate.bat（运行虚拟环境，在命令行也可以）

import sys   # 文件系统

print(sys.path)  # 保留python的路径   执行以上两行看到已经把虚拟环境加进来了，可以开始正式地编程了！每一个项目都需要开启自己的一个虚拟环境。
